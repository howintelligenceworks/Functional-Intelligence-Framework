# Functional-Intelligence-Framework
The Functional Intelligence Framework is an open scientific framework for understanding intelligence as a dynamic system that transforms information into adaptive performance.

## Vision

Our goal is to develop a transparent, testable, and cumulative scientific framework that advances the study of intelligence through open theory, empirical research, and collaborative science.

## Repository contents

- Framework documentation
- Research papers
- Figures
- Open resources
- Future datasets
- Assessment tools
- Prediction registry

This repository is under active development.
